# Changelog

All notable changes to the SPIDR framework will be documented in this file.
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-01

### Added
- Phase 1 Foundation: Monorepo workspace configuration.
- Standard analysis rules in `analysis_options.yaml`.
- GitHub Actions CI build definition.
- Package scaffolding for `spidr`, `spidr_core`, `spidr_browser`, `spidr_crawler`, `spidr_stealth`, `spidr_storage`, `spidr_proxy`, `spidr_ai`, and `spidr_cli`.
- Base architecture documentation, Roadmap, Contribution Guide, and Licenses.
