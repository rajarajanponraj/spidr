# spidr_stealth

The evasion and anti-detection spoofer plugin for the SPIDR framework. 

## Features

- **User-Agent Rotation**: Generates organic web browser identity configurations.
- **Canvas & WebGL Masking**: Modifies context profiles to prevent device fingerprinting.
- **Language & Timezone Spoofing**: Matches local runtime locations to proxy origins.
- **Header Randomization**: Re-orders request headers to match standard desktop/mobile browsers.
